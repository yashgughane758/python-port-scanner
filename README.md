# python-port-scanner
A simple multithreaded port scanner built using Python.

About the Project:
This project scans ports on a target IP address and shows which ports are open. It uses Python's socket and threading modules to perform the scan faster by running multiple threads at the same time.

Features:
Scan ports from 1 to 1024
Multithreaded scanning for better performance
Displays open ports
Shows total scan time

Technologies Used:
Python 3
Socket Module
Threading Module
